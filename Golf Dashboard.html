GolfCourseAPI (online course search) · DEV

Pull CR/Slope automatically from ~30,000 courses worldwide. Create a free account at golfcourseapi.com and paste your API key here. The key is stored locally in your browser only.

Default: https://api.golfcourseapi.com (often CORS-blocked).
With local proxy: http://localhost:8765
💡 Getting "Failed to fetch" or CORS errors? Run the bundled golf-proxy.py in a terminal and set Base URL to http://localhost:8765.

Export and import · DEV

All data is stored locally in your browser. Export to JSON for backup or moving to another machine.

🤝 Community courses: Export the courses you've contributed as a .json file. Send to a friend who can import them, or share on Discord/Reddit for the whole Fairway Rivalry community.

About WHS calculations

Score Differential = (113 / Slope) × (Score − Course Rating). A lower differential = better round.

Handicap Index = average of the 8 best differentials from your last 20 rounds. With fewer rounds, the WHS ladder applies: 3 rounds → lowest − 2.0, 4 → lowest − 1.0, 5 → lowest, and so on up to 20+ rounds.

Course Handicap = HCP Index × (Slope/113) + (CR − Par), rounded.

This dashboard uses gross score directly. For fully accurate WHS calculation, scores should be adjusted to Net Double Bogey per hole — this is done automatically when you log hole-by-hole.

A spectator gets the live scoreboard on their phone — no friction, just open the link. If you and your opponent both add the round through Match Mode, both must confirm the result before it counts toward the Season.

How co-signing works ▾
Two-person matches push the final result to both players' phones via Supabase Realtime. Each player taps Confirm — only then is the round saved to the league table. A solo logger can still finalize alone, but the result is marked unconfirmed on the table.
